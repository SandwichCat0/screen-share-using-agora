项目原理：
	1.打手红玩使用DMA或者双机挂，主机发起屏幕共享，客户远程观看
	2.客户和考官打微信语音，同时麦克风会通过本项目虚拟网卡传递到游戏内（三角洲声音->麦克风里面选择VB-OUTPUT）
	3.客户三角洲大厅挂机，游戏日志会自动从打手电脑传递到客户游戏路径下（注意本项目的fsutil.sys被我删除，这是后期更改$MFT时间的驱动，由于是实时传递日志，此项目不再需要，如果你不理解请无视）
	4.需要注意日志传递服务器需要你自己填，你也可以来找我一条龙全部搞好，中转服务器有很多开源项目，自己搞一个用就行了，需要更改protobuf head为客户硬件信息，本项目源码已揭示
